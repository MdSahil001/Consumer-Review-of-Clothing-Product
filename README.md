# Clothing Review Multilabel Classification

This project trains a **multilabel text classification model** on a dataset of customer clothing reviews.  
Each review can belong to multiple categories simultaneously ‚Äî such as *fit, material, comfort, fashion,* etc.

The dataset (Consumer Review of Clothing Product ‚Äì Fit, Feel, and Fashion: A Deep Dive into Clothing Reviews 2023) contains:

- Customer review text  
- Multiple labels per review  
- Various clothing-related categories  

This repository includes:
- A baseline deep learning model using **TensorFlow/Keras**
- Text preprocessing pipeline
- Multilabel classification training script
- Placeholder sample dataset (`data/sample.csv`)
